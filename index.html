  • 매체 객체 기반 공통 파이프라인 — API URI·응답 property 매핑·엑셀 컬럼·마크업 함수를 하나의 매체 객체로 정의하고 함수명을 일치시켜, 공통 파이프라인에서 매체 객체만 교체하면 동일하게 동작하는 구조로 재설계. 기존 if-else 분기 폭증 문제 해결.
  • 검색 UI 자동 구성 — 매체마다 검색 조건 항목 자체가 달라 매체 객체에 검색 조건을 키-값으로 정의하고, 키 유무에 따라 검색 UI가 자동 구성되도록 설계. 신규 매체 추가 시 검색 화면 별도 구현 불필요.
  • 공통화 사이드 이펙트 대응 — 신규 매체 추가 시 공통 구조 변경이 기존 매체에 사이드 이펙트를 유발하는 경험을 하면서, 이후 전체 매체 회귀 테스트 방식으로 검증 범위 확대. 이후 추가 오류 0건 유지.
